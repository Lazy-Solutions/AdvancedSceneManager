# Custom Nodes

There are two types of nodes: **FlowNode** and **DataNode**.

- **FlowNode**  
  Controls execution. Has `Flow.In`, `Flow.Out`, and any defined inputs/outputs.

- **DataNode**  
  Used to retrieve or provide data. Does not control flow.

# Extra Nodes

Nodes in this folder are typically user-created or more niche additions that do not fit in the main package.

> If you have useful nodes or ideas worth sharing, feel free to contribute!
